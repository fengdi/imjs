# Imjs
## A tiny javascript module loader

`im.js` is a simple solution to modular javascript development.

### Features
 * 简单的API
 * URI即Id
 * 异步多线程调用

### How to use
define(function(){
  //定义模块
});
require("foo",function(){
  //调用模块
});


For more advanced usage check the [`test.html`](https://github.com/fengdi/).
